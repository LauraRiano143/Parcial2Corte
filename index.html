¿Cómo funciona?

La captura de ECG usando un dispositivo electrónico tiene diferentes etapas, por lo tanto, para su correcto funcionamiento debes seguir estos pasos:

  1. Preparar el dispositivo
  2. Preparar el paciente
  3. Iniciar el proceso de captura
  4. Visualizar la señal ECG
  5. Guardar la señal obtenida
  6. Generar reporte

Sportholter es una aplicación web en donde podrás tener registro de la salud cardiovascular de personas deportistas.
